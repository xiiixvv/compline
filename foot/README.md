# Compline & Lauds for Foot

Dark and light colorschemes for [Foot](https://codeberg.org/dnkl/foot), based on monastic tradition.

## Installation

To install these themes, copy the files to your Foot configuration directory:

```sh
mkdir -p ~/.config/foot/themes
cp compline.ini ~/.config/foot/themes/
```

Include theme in your configuration in `~/.config/foot/foot.ini`:

```
include=~/.config/foot/themes/compline.ini
```

Then set your theme of choice in the same file:

```
initial-color-theme=1
```

Or:

```
initial-color-theme=2
```
