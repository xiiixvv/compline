# lauds-ice

A port of Lauds for IceWM

## Installation
```git clone https://github.com/joshuablais/compline.git```

Place ```compline/icewm/lauds``` into
```~/.icewm/themes/```
or
```/usr/share/icewm/themes/```

Apply the theme through the settings menu.

![Preview](preview.png)

### Notes
Terminal colors separate from this theme.
Font is the SauceCodePro Nerd Font
